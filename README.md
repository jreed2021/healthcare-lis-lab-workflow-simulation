# healthcare-lis-lab-workflow-simulation
Project Title

Laboratory Information System Workflow Simulation

Description

This project models the core workflow of a Laboratory Information System (LIS) similar to Epic Beaker. The goal is to demonstrate how laboratory tests, specimen labels, and workflow logic are configured and managed within healthcare information systems.

Components

The simulation includes:

• Lab test compendium (test builds)
• Specimen workflow mapping
• Barcode label logic
• Switch rule configuration
• Troubleshooting scenarios

Example Workflow
Physician Order
↓
LIS Receives Order
↓
Barcode Label Generated
↓
Specimen Collection
↓
Laboratory Instrument Testing
↓
Results Returned to LIS
↓
Results Posted to Patient Chart
Skills Demonstrated
